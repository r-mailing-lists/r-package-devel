# r-package-devel

Package development help

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-package-devel/](https://stat.ethz.ch/pipermail/r-package-devel/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics
